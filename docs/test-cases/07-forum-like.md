# TC-007: 点赞帖子

## 用例描述
验证用户点赞帖子的功能

## 前置条件
- 用户已登录
- 帖子存在

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 进入帖子详情页 | 帖子内容显示 |
| 3 | 找到点赞按钮 | 显示当前点赞数 |
| 4 | 点击点赞按钮 | 点赞成功，图标变化 |
| 5 | 验证点赞状态 | 点赞数+1，图标高亮 |

## 预期结果
- 点赞成功
- 点赞数正确增加
- 点赞状态正确显示

## 通过标准
- 点赞成功，无错误提示
- 点赞数正确更新
- UI状态正确变化
