# TC-003: 浏览帖子列表

## 用例描述
验证用户浏览论坛帖子列表功能

## 前置条件
- 系统存在已发布的帖子数据

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 访问 http://localhost:3000 | 首页加载成功 |
| 2 | 点击"论坛"Tab或导航 | 进入论坛页面 |
| 3 | 等待帖子列表加载 | 列表显示帖子卡片 |
| 4 | 滚动页面加载更多 | 更多帖子加载成功 |

## 预期结果
- 帖子列表正确显示
- 包含标题、作者、浏览量、发布时间等信息
- 分页功能正常

## 通过标准
- 页面无报错
- 帖子列表正确渲染
- 至少显示一条帖子数据
