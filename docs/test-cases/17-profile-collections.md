# TC-017: 查看我的收藏

## 用例描述
验证用户查看自己收藏的帖子和商品

## 前置条件
- 用户已登录
- 用户有收藏的帖子或商品

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 进入个人中心 | 头像/我的显示 |
| 3 | 点击"我的收藏" | 进入我的收藏页面 |
| 4 | 等待收藏列表加载 | 显示收藏的帖子/商品 |
| 5 | 检查收藏信息 | 显示标题、收藏时间 |

## 预期结果
- 收藏列表正确显示
- 包含帖子和商品收藏

## 通过标准
- 页面无报错
- 收藏列表正确显示
