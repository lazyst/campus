# TC-015: 查看我的帖子

## 用例描述
验证用户查看自己发布的帖子列表

## 前置条件
- 用户已登录
- 用户有发布的帖子

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 进入个人中心 | 头像/我的显示 |
| 3 | 点击"我的发布" | 进入我的帖子页面 |
| 4 | 等待帖子列表加载 | 显示用户发布的帖子 |
| 5 | 检查帖子信息 | 显示标题、发布时间 |

## 预期结果
- 帖子列表正确显示
- 只显示当前用户的帖子

## 通过标准
- 页面无报错
- 帖子列表正确显示
