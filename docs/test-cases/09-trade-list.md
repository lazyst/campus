# TC-009: 浏览商品列表

## 用例描述
验证用户浏览二手商品列表功能

## 前置条件
- 系统存在已发布的商品数据

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 访问 http://localhost:3000 | 首页加载成功 |
| 2 | 点击"闲置"Tab或导航 | 进入商品页面 |
| 3 | 等待商品列表加载 | 列表显示商品卡片 |
| 4 | 检查商品信息 | 显示标题、价格、图片 |
| 5 | 滚动加载更多 | 更多商品加载成功 |

## 预期结果
- 商品列表正确显示
- 包含标题、价格、图片等信息
- 分页功能正常

## 通过标准
- 页面无报错
- 商品列表正确渲染
- 至少显示一条商品数据
