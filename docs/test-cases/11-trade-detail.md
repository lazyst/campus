# TC-011: 商品详情查看

## 用例描述
验证用户查看商品详情的功能

## 前置条件
- 系统中存在已发布的商品

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 访问商品页面 | 商品列表显示 |
| 2 | 点击任意商品卡片 | 进入商品详情页 |
| 3 | 等待详情加载 | 页面显示商品完整信息 |
| 4 | 检查商品信息 | 显示标题、描述、价格、卖家信息、发布时间等 |
| 5 | 检查联系方式 | 显示卖家联系方式 |

## 预期结果
- 商品详情正确显示
- 包含完整的商品信息
- 卖家信息正确显示

## 通过标准
- 详情页无报错
- 内容完整显示
- 联系方式正确显示
