# TC-016: 查看我的商品

## 用例描述
验证用户查看自己发布的商品列表

## 前置条件
- 用户已登录
- 用户有发布的商品

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 进入个人中心 | 头像/我的显示 |
| 3 | 点击"我的闲置" | 进入我的商品页面 |
| 4 | 等待商品列表加载 | 显示用户发布的商品 |
| 5 | 检查商品信息 | 显示标题、价格、状态 |

## 预期结果
- 商品列表正确显示
- 只显示当前用户的商品

## 通过标准
- 页面无报错
- 商品列表正确显示
