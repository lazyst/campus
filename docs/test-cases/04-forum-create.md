# TC-004: 发布帖子

## 用例描述
验证用户发布论坛帖子的功能

## 前置条件
- 用户已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 点击底部Tab"论坛" | 进入论坛页面 |
| 3 | 点击右上角"发布"按钮（+） | 进入帖子发布页面 |
| 4 | 输入帖子标题：E2E测试帖子 | 标题输入框显示 |
| 5 | 点击版块选择器 | 弹出版块选择列表 |
| 6 | 选择一个版块（如：学习交流） | 版块显示已选择 |
| 7 | 输入帖子内容：这是E2E自动化测试帖子内容 | 内容文本框显示 |
| 8 | 点击"发布"按钮 | 发布成功，页面返回 |
| 9 | 验证帖子已发布 | 帖子列表显示新帖子 |

## 预期结果
- 帖子发布成功
- 页面返回论坛列表
- 新帖子显示在列表中

## 关键表单元素

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 帖子标题 | 输入框 | 是 | 5-30字 |
| 选择版块 | 选择器 | 是 | 点击弹出选择列表 |
| 帖子内容 | 文本域 | 是 | 最多5000字 |
| 添加图片 | 上传组件 | 否 | 最多9张 |
| 添加标签 | 输入框 | 否 | 输入后按回车 |

## 通过标准
- 发布成功，无错误提示
- 帖子正确保存到数据库
- 页面正确返回列表
