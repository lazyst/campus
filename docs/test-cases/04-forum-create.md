# TC-004: 发布帖子

## 用例描述
验证用户发布论坛帖子的功能

## 前置条件
- 用户已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 点击发布按钮（+图标） | 进入发布页面 |
| 3 | 选择板块 | 板块选中状态 |
| 4 | 输入标题：E2E测试帖子 | 标题输入框显示 |
| 5 | 输入内容：这是E2E自动化测试帖子 | 内容输入框显示 |
| 6 | 点击发布按钮 | 跳转至帖子详情页 |
| 7 | 验证帖子已发布 | 帖子内容正确显示 |

## 预期结果
- 帖子发布成功
- 跳转至帖子详情页
- 帖子内容正确显示

## 通过标准
- 发布成功，无错误提示
- 帖子正确保存到数据库
- 页面正确跳转
