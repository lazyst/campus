# TC-006: 评论帖子

## 用例描述
验证用户对帖子进行评论的功能

## 前置条件
- 用户已登录
- 帖子存在

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 进入帖子详情页 | 帖子内容显示 |
| 3 | 滚动到评论区 | 评论区可见 |
| 4 | 输入评论内容：E2E测试评论 | 评论输入框显示 |
| 5 | 点击发送按钮 | 评论发送成功 |
| 6 | 验证评论显示 | 评论出现在评论区 |

## 预期结果
- 评论发布成功
- 评论正确显示在评论区

## 通过标准
- 评论成功，无错误提示
- 评论正确保存和显示
