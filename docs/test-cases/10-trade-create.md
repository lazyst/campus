# TC-010: 发布商品

## 用例描述
验证用户发布二手商品的功能

## 前置条件
- 用户已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 点击发布按钮（+图标） | 显示发布选项 |
| 3 | 选择"闲置" | 进入商品发布页面 |
| 4 | 输入标题：E2E测试商品 | 标题输入框显示 |
| 5 | 输入描述：E2E自动化测试商品 | 描述输入框显示 |
| 6 | 输入价格：100 | 价格输入框显示 |
| 7 | 选择商品分类 | 分类选中 |
| 8 | 点击发布按钮 | 跳转至商品详情页 |
| 9 | 验证商品已发布 | 商品信息正确显示 |

## 预期结果
- 商品发布成功
- 跳转至商品详情页
- 商品信息正确显示

## 通过标准
- 发布成功，无错误提示
- 商品正确保存到数据库
- 页面正确跳转
