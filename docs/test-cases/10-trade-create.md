# TC-010: 发布商品

## 用例描述
验证用户发布二手商品的功能

## 前置条件
- 用户已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 点击底部Tab"闲置" | 进入商品页面 |
| 3 | 点击右下角"发布"按钮（+） | 进入商品发布页面 |
| 4 | 输入物品名称：E2E测试商品 | 名称输入框显示 |
| 5 | 选择闲置类型：点击"出售"按钮 | 出售按钮高亮 |
| 6 | 输入价格：100 | 价格输入框显示100 |
| 7 | 选择分类：点击"生活用品"按钮 | 分类按钮高亮 |
| 8 | 选择成色：点击"95新"按钮 | 成色按钮高亮 |
| 9 | 输入物品描述：E2E自动化测试商品描述 | 描述文本框显示 |
| 10 | 点击"发布"按钮 | 发布成功，页面返回 |
| 11 | 验证商品已发布 | 商品列表显示新商品 |

## 预期结果
- 商品发布成功
- 页面返回商品列表
- 新商品显示在列表中

## 关键表单元素

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 物品名称 | 输入框 | 是 | 最多50字 |
| 闲置类型 | 按钮 | 是 | 出售/求购，默认出售 |
| 价格 | 输入框 | 是 | 数字 |
| 分类 | 按钮组 | 是 | 电脑数码、学习资料、生活用品等 |
| 成色 | 按钮组 | 否 | 全新、99新、95新等，默认95新 |
| 物品描述 | 文本域 | 是 | 最多500字 |
| 上传图片 | 上传组件 | 否 | 最多9张 |
| 交易地点 | 输入框 | 否 | 文本 |

## 通过标准
- 发布成功，无错误提示
- 商品正确保存到数据库
- 页面正确返回列表
