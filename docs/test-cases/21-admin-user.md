# TC-021: 用户管理

## 用例描述
验证管理员管理用户的功能

## 前置条件
- 管理员已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录管理后台 | 进入仪表盘页面 |
| 2 | 点击"用户管理"菜单 | 进入用户管理页面 |
| 3 | 等待用户列表加载 | 显示用户列表 |
| 4 | 检查用户信息 | 显示用户名、手机号、状态 |
| 5 | 点击禁用/启用按钮 | 用户状态变更 |
| 6 | 验证状态更新 | 状态正确变化 |

## 预期结果
- 用户列表正确显示
- 禁用/启用功能正常

## 通过标准
- 页面无报错
- 用户列表正确显示
- 状态变更功能正常
