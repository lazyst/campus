# TC-022: 板块管理

## 用例描述
验证管理员管理论坛板块的功能

## 前置条件
- 管理员已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录管理后台 | 进入仪表盘页面 |
| 2 | 点击"板块管理"菜单 | 进入板块管理页面 |
| 3 | 等待板块列表加载 | 显示板块列表 |
| 4 | 点击"新增板块"按钮 | 弹出新增表单 |
| 5 | 输入板块名称：测试板块 | 名称输入框显示 |
| 6 | 输入板块描述：测试描述 | 描述输入框显示 |
| 7 | 点击保存 | 板块创建成功 |
| 8 | 验证板块显示 | 新板块出现在列表 |

## 预期结果
- 板块列表正确显示
- 新增板块功能正常

## 通过标准
- 页面无报错
- 板块列表正确显示
- 新增功能正常
