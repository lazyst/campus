# TC-023: 帖子管理

## 用例描述
验证管理员管理论坛帖子的功能

## 前置条件
- 管理员已登录
- 系统中存在帖子

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录管理后台 | 进入仪表盘页面 |
| 2 | 点击"帖子管理"菜单 | 进入帖子管理页面 |
| 3 | 等待帖子列表加载 | 显示帖子列表 |
| 4 | 检查帖子信息 | 显示标题、作者、板块 |
| 5 | 点击删除按钮 | 弹出确认框 |
| 6 | 确认删除 | 帖子删除成功 |
| 7 | 验证列表更新 | 帖子从列表移除 |

## 预期结果
- 帖子列表正确显示
- 删除功能正常

## 通过标准
- 页面无报错
- 帖子列表正确显示
- 删除功能正常
