# TC-024: 商品管理

## 用例描述
验证管理员管理二手商品的功能

## 前置条件
- 管理员已登录
- 系统中存在商品

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录管理后台 | 进入仪表盘页面 |
| 2 | 点击"商品管理"菜单 | 进入商品管理页面 |
| 3 | 等待商品列表加载 | 显示商品列表 |
| 4 | 检查商品信息 | 显示标题、价格、卖家 |
| 5 | 点击删除按钮 | 弹出确认框 |
| 6 | 确认删除 | 商品删除成功 |
| 7 | 验证列表更新 | 商品从列表移除 |

## 预期结果
- 商品列表正确显示
- 删除功能正常

## 通过标准
- 页面无报错
- 商品列表正确显示
- 删除功能正常
