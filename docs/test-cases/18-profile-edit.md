# TC-018: 资料编辑

## 用例描述
验证用户编辑个人资料的功能

## 前置条件
- 用户已登录

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 登录系统 | 进入首页 |
| 2 | 进入个人中心 | 头像/我的显示 |
| 3 | 点击头像或编辑资料 | 进入资料编辑页面 |
| 4 | 修改昵称：新昵称 | 昵称输入框显示新昵称 |
| 5 | 修改个人简介：新简介 | 简介输入框显示 |
| 6 | 点击保存按钮 | 保存成功 |
| 7 | 返回个人中心 | 显示更新后的资料 |

## 预期结果
- 资料编辑成功
- 资料正确更新显示

## 通过标准
- 保存成功，无错误提示
- 资料正确更新
