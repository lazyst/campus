# TC-019: 管理员登录

## 用例描述
验证管理员登录管理后台的功能

## 前置条件
- 管理员账号已存在：admin/admin123

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 打开浏览器，访问 http://localhost:3001 | 管理后台登录页加载 |
| 2 | 输入用户名：admin | 输入框显示用户名 |
| 3 | 输入密码：admin123 | 密码输入框显示****** |
| 4 | 点击登录按钮 | 页面跳转到仪表盘 |

## 预期结果
- 登录成功，页面跳转至仪表盘
- Token正确保存

## 测试数据
- 用户名：admin
- 密码：admin123

## 通过标准
- 登录成功，无错误提示
- 页面正确跳转
