# TC-005: 帖子详情查看

## 用例描述
验证用户查看帖子详情的功能

## 前置条件
- 系统中存在已发布的帖子

## 测试步骤

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 访问论坛页面 | 帖子列表显示 |
| 2 | 点击任意帖子卡片 | 进入帖子详情页 |
| 3 | 等待详情加载 | 页面显示帖子完整内容 |
| 4 | 检查帖子信息 | 显示标题、内容、作者、时间 |
| 5 | 检查评论区 | 显示已有评论 |

## 预期结果
- 帖子详情正确显示
- 包含完整的帖子信息
- 评论列表正确显示

## 通过标准
- 详情页无报错
- 内容完整显示
- 相关评论正确加载
